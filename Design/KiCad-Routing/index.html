Add Limits

  • https://www.wayneandlayne.com/blog/2013/02/26/kicad-tutorial-pcb-edges/

According to hackaday article:

  • Track Clearance - 15mil (0.381mm)
  • Min Track Width - 22mil (0.5588mm)
  • Min Drill size - 20mil (0.508mm)
  • Vias / Outer - Min 18mil (0.4572mm) / Max 20mil (0.508mm)
  • Pads top / bottom Min 15mil (0.381mm)
  • For autorouting set grid to 2mil (0.0508mm)

For KiCad this translates to

  • Clearance: 0.381
  • Track Width: 0.5588
  • Via Dia: 0.7
  • Via Drill: 0.508
  • uVia Dia: 0.7
  • uVia Drill: 0.508

Also make sure to apply the added Net Class to all the devices before exporting the .dsn for autorouting

for the board

  • 24 gauge wire or thinner is used for Vias
  • Drill bit is 0.02” (0.508mm) in diameter for via holes
  • Drill bit is 0.04” (1.016mm) in diameter for header pin holes
